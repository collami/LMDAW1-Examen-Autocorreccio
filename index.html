Información

La prestigiosa Universidad de Gromenauer ofrece un curso intensivo de javascript. Por ese motivo, abrimos nuestra página web para poder entrar en contacto con aquellos interesados.

El curso consta de diferentes niveles. Para determinar el nivel de los potenciales alumnos, se presenta una prueba tipo test que se debe realizar para acceder al curso.

A continuación, se detallan las instrucciones para completar el test.

Atte. La Directiva de la Universidad de Gromenauer.

Instrucciones

Este examen de prueba pondrá a prueba todos tus conocimientos.

Te encontrarás con diversas preguntas, unas solo tienen una respuesta posible a elegir entre varias; otras tendrás que responderlas escribiendo una única palabra y algunas tienen diversas respuestas correctas.

Debes responder OBLIGATORIAMENTE a todas las preguntas. No podrás corregir el examen si dejas alguna en blanco.

¡Suerte!
